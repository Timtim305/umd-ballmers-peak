Coding Question 1

Nick Smazbino is trapped in a computer! He is stuck in the heap space, and between him and freedom lies many arrays. To cross these arrays, our valiant hero Nick must abide by certain rules:

1. Given a value x in the array, Nick can only move a maximum of x steps forward into the array.

2. With each step, Nick can choose to either retain his previous amount of allocated steps, or reset and take the new value he is currently on as his maximum amount of available steps.

3. If Nick cannot surpass the end of the array, he loses.

Write a method where, given all of these arrays, determines whether or not Nick can reach or surpass the end. We have supplied the method header and example for you:

private boolean canNickPass(int[][] array) {}

Example (on a per-array basis):

Given [1, -2, 2] -> return False, given [2, 10, 0, -1] -> return True
